講師画面のデモ映像

　学習者のプレゼンテーションを映像化し, 講師画面を見ていただけるようにしました.
　学習者PCでは, PC画面にDAW画面を表示し再生します (プレゼンテーションのイメージ).
　本システムを使い指導者PCに送信し, 指導者PCで録画する (Media Recording API を利用).
　つまり, 本システムを使用しても, "このレベルのオーディオのクオリティは担保される" ということです.
　フルスクリーンするとPC上でイメージし易いかと思います.

デモ映像制作環境

　指導者PC　CPU: Core-i3 3.06GHz, メモリ: 12GB, OS: MacOSX 10.5
　学習者PC　CPU: Core-i3 2,66GHz, メモリ: 6GB, OS: Windows7 Pro 64bit, DAW: SONAR X3
　ネットワーク: 1GbpsのLAN環境.
　Webサーバ: 指導者PCに機能をが持つ.
　シグナリング/ICEサーバ: SkyWayを利用.
